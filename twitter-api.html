Using the Twitter API

In this tutorial we will make a twitter bot which tweets a certain message when we run our nodejs application. The purpose of this tutorial is to understand how a twitter API can be used for different functionalities.

Language/Framework used: Javascript, NodeJS

Last Updated: 15-03-2022

Link to entire code

https://github.com/TanishqNagpal/twitterAPI

Step -1

For getting started with using the twitter api, first go to developer.twitter.com and then click on Developer Portal. Under Projects and Apps, make a new project.

Then in User Authentication settings give acces to OAuth 2.0 and OAuth 1.0a. Choose the option of Automated App or Bot in "Type of App". At last give the Read and Write permission.

Note: These settings are in accordance with our use of making a bot for tweeting. You can change these settings according to your use of the twitter API.

Now on the Twitter Developers Dashboard, you have to "Apply for Elevated" option to get higher level access.

After saving all these changes, regenrate the API keys and save it.

Step -2

Now make a new project with the name twitter-bot. Inside the terminal run the following command for adding node modules.

npm init -y

Run the following command in the terminal to add the twitter module which we will be using in this bot. Refer link for documentation regarding this module.

npm i twitter-api-v2

NOTE: There are many such modules for different use cases.

Step -3

Make a file twitterClient.js and require the twitter api module.

Then instantiate a twitter client and add your own API keys, secret and access token and secret generated from the dashboard to the following code. 

          
            const {TwitterApi}=require("twitter-api-v2")

            const client=new TwitterApi({
            
                appKey: "#",
                appSecret: "#",
                accessToken: "#",
                accessSecret: "#"
            })

Step -4

Create another client with read write permission and then export it so that it can be used in the nodejs application. 

          
            const {TwitterApi}=require("twitter-api-v2")

            const client=new TwitterApi({
            
                appKey: "",
                appSecret: "",
                accessToken: "",
                accessSecret: ""
            })
            
            const rwClient=client.readWrite
            
            module.exports=rwClient

Step -5

Make another file index.js and import the twitter client just created. 

          
            const rwClient =require("./twitterClient.js");

Step -6

Create an asynchronous function for the tweet functionality which makes the client to tweet the message.

Then call the tweet function.

Run the application using node index.js command in your terminal and you will see the message being tweeted from your account without actually opening the account. This concludes how we can use services or data from twitter without actually using the account. 

        
            const rwClient =require("./twitterClient.js");

            const tweet =async () =>{
            
                try{
            
                    await rwClient.v1.tweet("app works fine")
            
            
                }catch(e) {
            
                    console.error(e)
                }
            }
            
            
            tweet()

NOTE: If you run the application again instantaneously with the same message as the tweet then it will give an error as twitter considers duplica messages done in a short period of time as spam.

NOTE: This tutorial was just one way in which you can access the twitter api and make use of it according to the functionality. There are various other functionalities that can be implemented using the API like seeing the top trends on twitter, gathering information from tweets about any topic, etc.
