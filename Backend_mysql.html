Integrating a Payment Gateway using Razorpay

How many times have you went to your favourite e-commmerce or food ordering website and ordered using payment options like UPI, Debit Card, Credit Card, etc ? With this tutorial you can implement the same in your own web-app.

Language/Framework used: Javascript, NodeJS

Step -1

First of all you have to install MySQL in your device. Download it from the official website.

Now, After setting up the environmental variables, You need to install a package named mysql which is used to connect mysql database with your node js app.

Run the following command in the terminal: npm install mysql

Now copy the following code snippet to connect your application with MySQL database.

            
                var express = require(‘express’);
                var mysql = require(‘mysql’);

                var connection = mysql.createConnection({
                host: ‘localhost’ ,
                user: ‘root’,
                password: ‘’ ,
                database: ‘database-name’
                });   

                connection.connect(function(error) {
                    if(!!error){
		            console.log(‘Error’);
	                } else {
		            console.log(‘Connected successfully.’);
                    }
                });

            });

Note: Make sure you have already create a database in your system. After creating the database enter it's name properly in the code.
