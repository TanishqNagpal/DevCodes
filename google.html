Google Authentication

In this tutorial you will able to create a google sign in button easily. Just follow this simple step and the google button will be on your page.

Language/Framework used: Javascript, HTML

Last Updated: 20-03-2022

Step - 1

Create a new file google.html and add the following code to setup a basic html file template. 

          
            <html>

            <head>
            	<title>Google login</title>
            </head>
            
            <body>
            
            	<h3> Welcome to google authentication tutorial.</h3>  
            	
            </body>
       
            </html>

Step - 2

Now, for accessing the google APIs you need to generate your client id from the official website https://console.cloud.google.com

If you don't know how to get your client id you can visit this Link for the referance.

After generating your client id, you need to add this two line in your html file under head section. 

          
            <meta name="google-signin-client_id" content="1071066717069-6d3h3m8no5s9lgdr1pp8l37h8q8js15n.apps.googleusercontent.com">
            	<script src="https://apis.google.com/js/platform.js" async defer></script>

Note: You need to replace the client id in the content with your respective client id.

Step - 3

Now, to put the Google button to your webpage, google has its own class so we just need to add that class in our body section of html file. 

          
            <div class="g-signin2" data-onsuccess="signIn"></div>

Here, we have created a signIn function to check whether we are successfully logged in or not. If it is successfull we will get the details of that user on the console.

so, copy the following script code and paste it after the body section ends. 

        
          <script>
          	function signIn(googleUser){
          		console.log('User is ' + JSON.stringify(googleUser.getBasicProfile()))
          	}

          </script>

Step - 3

Now, we will add one sign out button for signing out the user after logging in.

So, copy the foolowing line and add it in to the body section. 

          
            <button onclick="signOut()">Sign out</button>

Here, we have created a signOut function to check whether we are successfully logged out or not. If it is successfull we will get the message on the console.



so, copy the following code and paste it in the script tag. 

      
        function signOut(){
          gapi.auth2.getAuthInstance().signOut().then(function(){
            console.log("User signed out successfully.")
          })
        }

Step - 4

For running this file, you need to create a simple http server. So, go to terminal and run the following command to create a simple http server. 

        
          python3 -m http.server 3000

Note: Here, you need to set the port number same as you have set while creating your google client id.
