Google Authentication

In this tutorial you will able to create a google sign in button easily. Just follow this simple step and the google button will be on your page.

Language/Framework used: Javascript, React

Last Updated: 25-03-2022

Step - 1

First of all, you have to create a react application. For that run the following command in the terminal.
npm install -g create-react-app
npx creat-react-app demo-app

Step - 2

Now, Go to your project directory and open app.js file and replace the existing code with the following code. 

            
                import React, { Component } from 'react'
                import GoogleLogin from 'react-google-login'
                export class App extends Component {

                responseGoogle=(response)=>{
                console.log(response);
                console.log(response.profileObj);
                }

                render() {
                return (
                
                <div>
                    <GoogleLogin
                    clientId="1071066717069-6d3h3m8no5s9lgdr1pp8l37h8q8js15n.apps.googleusercontent.com"
                    buttonText="Login"
                    onSuccess={this.responseGoogle}
                    onFailure={this.responseGoogle}
                    cookiePolicy={'single_host_origin'}
                    >
                </div>

                )           
                }
                }

                export default App

Step - 3

Now you can see the google sign-in button after running your file on port 3000 (by default).
